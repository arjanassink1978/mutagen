What is Mutagen?

Mutagen automates API testing for Spring Boot projects. It discovers REST endpoints, generates RestAssured tests, runs mutation testing to detect weak spots, and opens a PR/MR with improved tests.

Quick Start

Bring your own key (BYOK):

export ANTHROPIC_API_KEY=sk-ant-...
# or
export OPENAI_API_KEY=sk-...
